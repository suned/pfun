    pfun.trampoline.Trampoline dataclass

    Base class for Trampolines. Useful for writing stack safe-safe recursive functions.

    pfun.trampoline.for_each(f, iterable)

    Map each in element in iterable to an Trampoline by applying f, combine the elements by and_then from left to right and collect the results

    Examples:

    >>> map_m(Done, range(3)).run()
(0, 1, 2)

    pfun.trampoline.filter_(f, iterable)

    Map each element in iterable by applying f, filter the results by the value returned by f and combine from left to right.

    Examples:

    >>> filter_m(lambda v: Done(v % 2 == 0), range(3)).run()
