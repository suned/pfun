## Install

`pip install pfun`

## Development

Requires [poetry](https://poetry.eustace.io/)

- Install dependencies with `poetry install`
- Build documentation with `poetry run sphinx-build -b html docs/source docs/build`
- Run tests with `poetry run pytest`
